¿Quiénes somos?

Soy Katia, una alumna del máster de CICE Desarrollo Web Full Stack, y este es un proyecto para demostrar que he aprendido todo el html, css y javascript que debería.

¿Puedes explicar lo que es esta web?

El proyecto consiste en utilizar una API gratuita cualquiera para obtener datos mediante la función fetch, mostrar dichos datos en pantalla, y que el usuario pueda elegir cuáles son sus favoritos. Los favoritos deberán guardarse en una lista donde se puedan almacenar datos persistentes, por lo tanto se debe usar localStorage. Por último, desde la sección de favoritos se pueden consultar los datos añadidos, así como eliminarlos. En mi caso es escogido una API de videojuegos por razones obvias, es mi hobby número 1 y en cuanto me dan a escoger un tema libre, mi tema elegido suele ser videojuegos :)

¿Qué te ha parecido el proyecto?

He disfrutado mucho realizando esta web (en ocasiones me he sentido workaholic incluso), no es postureo barato, realmente lo he hecho. Tenía algo de experiencia en Desarrollo web, pero muchos de esos conocimientos se ha quedado enterrados. Este proyecto me ha ayudado muchísimo a refrescar algunos conceptos y a aprender de cero otros muchos. por otra parte, también me siento humilde, pienso que construir una web como esta es apenas un 5% (o menos) de la profesión, con lo cual tengo todo por aprender. Pero me está gustado mucho (volver a) recorrer este camino :)
